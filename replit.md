# Project Overview
React Native Business Management Suite with a light theme and modular components.

## Architecture
- **Framework**: Expo (Router)
- **Styling**: React Native StyleSheet, Expo Linear Gradient
- **Theme**: Light mode with primary (`#1C439C`), secondary (`#E7B238`), and accent (`#FEED9D`) colors.

## Key Components
- `components/common/CustomAlert.tsx`: Reusable modal-based alert.
- `components/common/ScreenContainer.tsx`: Reusable screen wrapper with gradient background.

## Color Palette
- Primary: `#1C439C` (Blue)
- Secondary: `#E7B238` (Gold)
- Accent: `#FEED9D` (Light Gold/Yellow)
- Background: `#F8F9FA`
- Surface: `#FFFFFF`
